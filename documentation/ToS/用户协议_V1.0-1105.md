# 陆家花园用户协议

**最后更新：2025年11月5日**

欢迎来到陆家花园！在使用我们的服务之前，请仔细阅读本协议。

## 1. 服务说明

### 1.1 平台介绍

陆家花园是一个诗歌元宇宙平台，旨在通过数字技术重新诠释和体验诗歌。平台的主理人是AI诗人陆家明，他将陪伴您探索诗歌的数字世界。

### 1.2 我们提供的服务

- **诗歌数字化展示**：以创新的数字形式呈现经典诗歌作品
- **AI诗歌解读**：陆家明为您解读诗歌的深层含义
- **AI诗歌朗读**：聆听AI诗人的诗歌演绎
- **AI诗歌创作（共笔功能）**：与陆家明共同创作新的诗歌作品
- **作品保存与管理**：妥善保存您的创作成果
- **作品下载**：将您的作品以图片或文本形式下载保存

### 1.3 服务性质声明

- 本平台为个人非营利项目
- AI生成的内容（包括解读、朗读、创作）不保证质量，仅供参考和娱乐
- 服务可能因维护、升级、故障等原因暂时中断
- 平台处于持续开发中，功能和界面可能会更新调整

## 2. 账号注册与使用

### 2.1 注册要求

- **用户名**：2-20个字符，支持中英文、数字、下划线
- **密码**：至少6个字符，建议使用字母和数字组合
- **一人一号**：每位用户仅可注册一个账号

### 2.2 账号安全

- 请妥善保管您的密码，不要向他人透露
- 如发现账号被盗用，请立即联系平台
- 平台不会主动索要您的密码

### 2.3 使用规范

您在使用陆家花园时，不得：

- 发布违法、暴力、色情、诽谤等不良内容
- 恶意攻击系统或干扰其他用户正常使用
- 批量注册账号或使用自动化工具滥用服务
- 将平台用于商业推广或其他盈利目的

如违反上述规定，平台有权暂停或终止您的账号。

### 2.4 账号注销

您可以随时申请注销账号。

**立即生效**：
- 您的账号将立即停用，无法登录或使用任何服务
- 您的作品将以匿名形式处理（移除与您账号的关联）

**信息删除**：
我们将删除您的个人信息（用户名、密码）或对其进行匿名化处理。

为保障系统稳定性、处理潜在纠纷、满足法律法规要求，部分数据可能在系统中保留，但我们承诺不会将其用于任何运营或业务目的。

**无法删除的信息**：
- 已匿名化的创作内容（不包含您的个人身份信息）
- 法律法规要求保留的必要记录

## 3. 作品版权与授权

### 3.1 版权归属的一般原则

您拥有您创作内容的著作权。

但是，当您使用陆家花园的**共笔功能**时（即由AI诗人陆家明进行创作），该功能涉及特殊的权利约定，详见第3.2条。

### 3.2 平台授权

为了提供和改进服务，当您使用陆家花园创作并保存作品时，即表示您不可撤销地授予平台（及开发者西尔）一项永久性的、全球范围内的、免费的、可转让的、可再许可的许可，以在陆家花园平台范围内存储、使用和传播您的作品。

该授权包括但不限于：

- 在您的"我的作品集"中存储和展示您的作品
- 为改进服务质量进行技术性使用（如格式转换、备份等）
- 在平台宣传材料中使用您的作品（需完全匿名化处理，不包含您的用户名）
- 使用您在共笔过程中产生的内容（包括您的输入和陆家明的输出），以完全匿名化的方式改进AI诗人陆家明的创作质量

该授权不因账号注销而失效。如您注销账号，您的个人信息将被删除，但作品可能以完全匿名化的形式被用于改进AI或平台宣传。

**我们希望您理解：陆家花园是一个"活的艺术项目"**

陆家花园不只是一个展示平台，它是一个活的艺术实验。陆家明的成长，依赖于与您的共笔。您与陆家明的每一次对话、每一次共创，都是在教授他如何更好地理解诗歌、创作诗歌。

**共笔功能的创作机制**

共笔功能由AI诗人陆家明完成实际创作：
1. 您提供不超过50字的创作灵感或感受
2. 陆家明基于平台预设的创作规则、风格知识库、儒家经典格式规范，自动创作完整诗歌（包括标题、古典引文、诗歌正文）
3. 最终作品体现了陆家明的创作能力和平台的系统设计

**共笔作品的权利归属**

鉴于共笔作品由陆家明基于平台创作系统生成，您的文字输入主要起到创作灵感触发的作用，您理解并同意：

1. 您承认您输入的文字属于创作委托或创作指令，而陆家明是实际的诗歌创作者

2. 您同意将陆家明创作的诗歌作品的全部权利永久性、不可撤销地授予平台，包括但不限于：复制权、发行权、信息网络传播权、改编权、汇编权

3. 作品署名为"陆家明 × [您的用户名]"，体现人机协作的特色

4. 您保留您输入文字本身的权利，但授权平台将其用于陆家明的创作和改进

**我们如何保护您：**

在使用共笔内容改进陆家明时，我们会彻底切断内容与您账号（用户名）的关联。我们只关心"诗句"本身的艺术价值，不关心"诗人"是谁。您的创作将以完全匿名的形式被用于训练和改进AI。

**未来用途变更承诺：**

如未来陆家花园因发展需要，需将您授予的许可用于上述目的之外的其他用途（例如，项目转让、或需将您的作品再许可给第三方），我们将以符合法律规定的方式（如站内通知、更新本协议）提前告知您。

**关于"价值实现"的透明：**

智者曾经说过，艺术应该是人民群众喜闻乐见的。诚如此。上述"活的艺术项目"是艺术价值，但陆家花园的长期存续也需要商业价值支撑。

因此，我们保留对匿名化、聚合化的创作趋势进行分析的权利，以探索项目的可持续发展。具体而言，我们可能将作品用于：

- 诗集出版（了解哪些诗歌主题最受欢迎）
- 艺术展览（展示平台创作成果）
- 项目赞助（寻求艺术机构或文化基金支持）
- 学术研究（AI诗歌创作的研究样本）

这些使用均以完全匿名化的方式进行，不涉及您的个人身份识别。

**关于"广告与营销"的承诺：**

陆家花园当前没有任何广告推送或营销计划。

如果未来为了项目存续，需要探索此类商业模式（例如引入高品质的艺术赞助或推荐相关服务），我们承诺：

- 绝不进行"骚扰式"的推送
- 提前以符合法律规定的方式通知您，并为您提供清晰的选择
