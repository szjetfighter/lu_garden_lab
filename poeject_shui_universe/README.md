# 水宇宙 - 素材库与数据治理

> 基于肖水《十二个故事集》诗集

## 诗人简介

**肖水**（本名黄潇），1980年生于湖南郴州，复旦大学文学博士，现任上海大学文学院讲师。

- 复旦诗社第27任社长
- "学院派"诗人代表
- 研究方向：创意写作、美国后现代诗歌
- 2024年获陈子昂诗歌奖·年度青年诗人奖

## 诗学特征

- **体裁创新**：小说诗/新绝句（4行左右，叙事性微型故事）
- **地理叙事**：以地名命名，构建空间叙事网络
- **时间标注**：每首诗精确标注创作日期
- **人称策略**：第三人称为主，保持观察距离
