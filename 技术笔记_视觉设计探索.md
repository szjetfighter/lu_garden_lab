## 1.2 玻璃态设计（Glassmorphism）

### 核心实现

```css
.universe-card {
  background: linear-gradient(135deg, rgba(255, 255, 255, 0.4) 0%, rgba(248, 250, 252, 0.6) 100%);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow: 
    0 8px 32px rgba(0, 0, 0, 0.1),
    0 2px 8px rgba(0, 0, 0, 0.05),
    inset 0 1px 0 rgba(255, 255, 255, 0.6);
}
```

### 设计要素

| 要素 | 实现 | 作用 |
|------|------|------|
| 半透明渐变 | `linear-gradient` + `rgba` | 基础玻璃质感 |
| 模糊效果 | `backdrop-filter: blur(10px)` | 磨砂玻璃效果 |
| 边框高光 | `border: 1px solid rgba(255,255,255,0.3)` | 边缘光泽 |
| 内阴影 | `inset 0 1px 0 rgba(255,255,255,0.6)` | 顶部高光 |
| 外阴影 | 多层box-shadow | 悬浮层次感 |

## 1.3 卡片背景贴图方案（已验证）

### 需求场景
- L0 Portal宇宙卡片需要展示各宇宙的视觉标识
- L1 模块卡片需要增强氛围感
- 要求保持玻璃态效果，不破坏文字可读性

### 技术方案：玻璃态 + 底图叠加

**核心CSS**：
```css
.universe-card.has-bg-image {
  background: 
    linear-gradient(135deg, rgba(255, 255, 255, 0.65) 0%, rgba(248, 250, 252, 0.75) 100%),
    var(--card-bg-image) center/cover no-repeat;
  background-size: auto, cover;
  background-position: center, center;
}
```

**组件接口**：
```typescript
interface Props {
  universe: Universe
  disabled?: boolean
  index?: number
  backgroundImage?: string  // 新增：背景图URL
}

// 动态样式
const cardStyle = computed(() => {
  const style: Record<string, string> = {
    animationDelay: `${props.index * 0.1}s`
  }
  if (props.backgroundImage) {
    style['--card-bg-image'] = `url(${props.backgroundImage})`
  }
  return style
})
```

**调用方式**：
```vue
<UniverseCard
  :universe="universe"
  :background-image="getCardBackgroundImage(universe.id)"
/>
```

### 验证结果（2025-12-11）

| 测试项 | 结果 |
|--------|------|
| 图片不拉伸 | ✅ `cover`模式居中裁剪 |
| 玻璃态效果 | ✅ 底图被半透明层覆盖 |
| 文字可读性 | ✅ 65%-75%透明度保证对比度 |
| 响应式 | ✅ 不同尺寸下表现一致 |

## 1.4 透明度调节参考

| 场景 | 推荐透明度 | 说明 |
|------|------------|------|
| 纯色/简单背景 | 0.4-0.6 | 当前默认值 |
| 复杂/高对比背景图 | 0.65-0.75 | 已验证值 |
| 深色主题 | 0.3-0.5 | 需单独测试 |

透明度过低会导致底图过于突出，影响文字阅读；过高则底图几乎不可见。
