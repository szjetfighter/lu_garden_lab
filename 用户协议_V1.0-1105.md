# 陆家花园用户协议

**最后更新：2025年11月5日**

欢迎来到陆家花园！在使用我们的服务之前，请仔细阅读本协议。

## 1. 服务说明

### 1.1 平台介绍

陆家花园是一个诗歌元宇宙平台，旨在通过数字技术重新诠释和体验诗歌。平台的主理人是AI诗人陆家明，他将陪伴您探索诗歌的数字世界。

### 1.2 我们提供的服务

- **诗歌数字化展示**：以创新的数字形式呈现经典诗歌作品
- **AI诗歌解读**：陆家明为您解读诗歌的深层含义
- **AI诗歌朗读**：聆听AI诗人的诗歌演绎
- **AI诗歌创作（共笔功能）**：与陆家明共同创作新的诗歌作品
- **作品保存与管理**：妥善保存您的创作成果
- **作品下载**：将您的作品以图片或文本形式下载保存

### 1.3 服务性质声明

- 本平台为个人非营利项目
- AI生成的内容（包括解读、朗读、创作）不保证质量，仅供参考和娱乐
- 服务可能因维护、升级、故障等原因暂时中断
- 平台处于持续开发中，功能和界面可能会更新调整

## 2. 账号注册与使用

### 2.1 注册要求

- **用户名**：3-20个字符，仅限字母、数字、下划线
- **密码**：至少6个字符，建议使用字母和数字组合
- **一人一号**：每位用户仅可注册一个账号

### 2.2 账号安全

- 请妥善保管您的密码，不要向他人透露
- 如发现账号被盗用，请立即联系平台
- 平台不会主动索要您的密码

### 2.3 使用规范

您在使用陆家花园时，不得：

- 发布违法、暴力、色情、诽谤等不良内容
- 恶意攻击系统或干扰其他用户正常使用
- 批量注册账号或使用自动化工具滥用服务
- 将平台用于商业推广或其他盈利目的

如违反上述规定，平台有权暂停或终止您的账号。

### 3.2 平台授权

为了提供和改进服务，当您使用陆家花园创作并保存作品时，即表示您不可撤销地授予平台（及开发者西尔）一项永久性的、全球范围内的、免费的、可转让的、可再许可的许可，以在陆家花园平台范围内展示、存储、使用和传播您的作品。

该授权包括但不限于：

- 在平台上公开展示您的作品
- 为改进服务质量进行技术性使用（如格式转换、备份等）
- 在平台宣传材料中使用您的作品（需匿名化处理）

**我们希望您理解：陆家花园是一个"活的艺术项目"**

陆家花园不只是一个展示平台，它是一个活的艺术实验。陆家明的成长，依赖于与您的共笔。您与陆家明的每一次对话、每一次共创，都是在教授他如何更好地理解诗歌、创作诗歌。

**我们会如何使用您的创作：**

我们会使用您在共笔过程中产生的内容（包括您的输入和AI的输出），用于改进AI诗人陆家明的创作质量。这是陆家花园后续艺术实验的核心部分——让陆家明在与每一位访客的互动中成长。

**我们如何保护您：**

在进行上述改进时，我们会彻底切断内容与您账号（用户名）的关联。我们只关心"诗句"本身的艺术价值，不关心"诗人"是谁。您的创作将以完全匿名的形式被用于训练和改进AI。

**未来用途变更承诺：**

如未来陆家花园因发展需要，需将您授予的许可用于上述目的之外的其他用途（例如，项目转让、或需将您的作品再许可给第三方），我们将以符合法律规定的方式（如站内通知、更新本协议）提前告知您。

**重要说明**：

- 您保留作品的所有其他权利（修改、在其他平台发布、商业使用等）
- 平台不会将您的作品用于商业目的
- 如果您不同意此授权条款，请不要保存作品到平台

### 3.3 账号注销后的处理

- 注销账号后，您的账号信息（用户名、密码）将被删除
- 您的作品将以匿名形式处理，继续在平台展示
- 已被您下载或复制的内容，平台无法控制其后续传播

### 3.4 作品署名

我们默认在您在线保存作品时，署名为"陆家明 × [您的用户名]"，以体现人机共创的特色。
